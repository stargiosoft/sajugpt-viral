# 바이럴 기능 요구사항 정의서 (PRD)

## 프로젝트 개요

- **기능명**: 주가 조작단 — "연애 시장 내 값어치는?"
- **바이럴 공식**: [주식 투자/시라노 연애 조작단 A] + [사주 기반 연애 주가 조작 작전 B] = "이런 건 처음이다"
- **바이럴 기능 간단 요약**: 사용자의 사주를 입력하면 연애운을 **주식 시세**로 환산한 종목 리포트 카드가 즉시 생성된다. 핵심 역설은 **"적정가는 20,000원인데 현재 시장가는 1,200원밖에 안 한다"** — 펀더멘털 대비 극심한 저평가. 이 바닥 주가를 올리기 위해 **주가 조작단** 3명이 소집된다. 공격파 본부장(당장 상장하라), 가치파 분석가(자기 가치 인식부터), 타이밍파 전략가(7월까지 기다려라) — 세 캐릭터가 서로 **토론하고 싸우면서** 유저의 연애 문제를 진단하고 주가 상승 작전을 설계한다. 유저는 턴마다 누구 편을 들지 선택하며, 선택이 최종 **작전 계획서**에 반영된다. 작전은 3단계(자기 가치 인식 → 모멘텀 확보 → 급등 트리거)로 구성되지만, **3단계 상세 전략은 잠겨 있다** — "작전 실행은 조작단원과 1:1로." 이 미완의 작전이 캐릭터 챗봇으로의 전환 호기심을 만든다. 결과물은 **종목 리포트 카드 1장**(현재가 + 적정가 + 저평가율 + 투자의견) — "내 연애 적정가 2만원인데 시장가 1,200원이래ㅋㅋ 저평가 국밥주" 가 대화를 만들고, **커플 M&A 시너지 분석**이 강제 2인 루프를 만든다.

## 바이럴 루프

### 바이럴을 일으키는 구조

1. **입력**: 생년월일시 + 현재 연애 상태 입력 (10초, 가입 불필요) → "사주증권 리서치센터 분석 중..." 로딩 연출
2. **종목 리포트 카드**: 현재가 + 적정가 + 저평가율 + 투자의견이 담긴 카드 1장 즉시 생성 → **바이럴 지점 1**
3. **주가 조작단 소집**: "이 주가, 조작 안 하면 상폐됩니다" → 3명의 조작단원이 소집되어 턴제 토론 시작
4. **작전 회의** (턴제 시뮬레이션): 3캐릭터가 서로 싸우며 유저의 문제를 진단 + 유저가 편을 드는 선택 (4~5턴)
5. **작전 계획서**: 3단계 주가 상승 작전 → 단, 3단계(급등 트리거)는 잠금 → **호기심 극대화**
6. **M&A 시너지 분석**: 상대방 생년월일 입력 → 합병 시너지 분석 → **강제 2인 루프**
7. **전환**: "작전 실행하기" → 조작단원 캐릭터 챗봇 1:1 유료 상담

### 유발 감정

| 감정 | 메커니즘 |
|------|---------|
| **뜨끔 + 자조 (Hook)** | "현재가 1,200원"이라는 숫자가 내 연애 현실을 적나라하게 보여줌 → "ㅋㅋ 맞는데" 캡처 |
| **위로 + 자존감 (Relief)** | "적정가 20,000원" — 내 진짜 값어치는 높은데 시장이 못 알아본 거라는 면죄부 |
| **몰입 + 재미 (Engagement)** | 3캐릭터가 서로 싸우는 걸 보면서 "ㅋㅋ 맞아 저 말이 맞는데" 편들기 |
| **찔림 + 인식 (Awareness)** | 토론 과정에서 내가 왜 저평가인지 원인이 하나씩 드러남 — 문제 인식 |
| **호기심 (Curiosity)** | 3단계 작전이 잠겨 있음 → "급등 트리거가 뭔데?" → 챗봇 전환 동기 |
| **비교 (Envy)** | 현재가 + 적정가 숫자 비교 → "너는 얼마야?" "적정가 얼마래?" |

### 논란 설계

- **조작단원 간의 충돌이 논란의 핵심 엔진**: 공격파 "당장 소개팅 나가세요" vs 가치파 "자존감도 안 올렸는데?" → 유저가 편을 들면서 자연스럽게 자기 문제를 인식
- **"현재가 800원"이 밈**: "상폐 직전" "관리종목" "동전주" → 자조적 공유 폭발
- **작전 계획서의 3단계 잠금이 안전장치**: 문제만 드러내고 끝나지 않음 — "해결 방법이 있는데 아직 못 봤다"는 미완의 구조가 불쾌감 대신 호기심을 만듦
- **스윗 스팟**: 캐릭터끼리 싸우는 거라 유저를 직접 공격하지 않음. 유저는 관전자+심판 포지션 — 찔리지만 안전함

## 사용자 스토리

**As** (누가): 사회적 스펙과 커리어는 탄탄하지만 외모 컴플렉스로 연애 시장에서 저평가받아 온 3040 여성 유저는

**Want to** (무엇을 원한다): 내 연애 주가가 왜 바닥인지 조작단원들의 토론을 보며 재밌게 진단받고, 주가를 올릴 구체적 작전을 받아서, 종목 리포트 카드를 친구들과 공유하며 "너는 현재가 얼마야?" 대화를 하기를 원한다.

**So that** (어떤 목적/가치를 위해): "적정가는 2만원인데 시장이 못 알아본 거"라는 숫자로 자존감을 회복하고, 조작단의 작전으로 "내가 뭘 바꿔야 하는지" 방향을 잡고, 잠긴 3단계(급등 트리거)가 궁금해져서 캐릭터 챗봇과 1:1로 작전을 실행하기 위해.

## MVP 필요 기능

### Step 1. 초저마찰 랜딩 및 정보 입력

- 메인 화면: **"연애 시장 내 값어치는?"** 헤드라인 + "사주증권 리서치센터가 당신의 연애 주가를 조회합니다" 서브카피 + 증권 HTS 스타일 다크 UI + **"시세 조회"** 버튼
- 입력폼: 생년월일(양력/음력), 성별, 태어난 시간(선택), 현재 연애 상태(싱글/썸/연애중). **10초 컷** 설계
- 입력 직후 **"사주증권 리서치센터 분석 중..."** 로딩 연출:
  - "종목 데이터 수집 중..."
  - "원국 밸류에이션 산정 중..."
  - "도화살 모멘텀 분석 중..."
  - "주가 조작단 소집 중... 📊"
  - → 증권사 느낌의 긴장감 빌드업 (2~3초)

### Step 2. 종목 리포트 카드 생성 (바이럴 핵심 — 1장 캡처 공유)

사주 분석 결과를 기반으로 증권사 리서치 리포트 형태의 카드 1장이 즉시 생성된다. **이 카드가 바이럴의 최소 단위.**

#### 2-1. 투자의견 배정 — 사주 요소와 연결

사주 원국에서 현재 연애운 상태를 추출하여 5단계 투자의견 중 1개를 자동 배정:

| 투자의견 | 사주 트리거 | 의미 | 카드 표시 |
|---------|-----------|------|----------|
| **강력 매수** | 도화살 2개 이상 + 현재 억압(편인 과다) | 매력 폭발 직전, 지금이 바닥 | 🔴 빨간 뱃지 |
| **매수** | 도화살 1개 + 정관/정인 동주 | 안정적 상승 예상 | 🟠 주황 뱃지 |
| **보유(존버)** | 식신 보유 + 현재 대운 평이 | 급등은 아니지만 손절 금지 | 🟡 노랑 뱃지 |
| **비중 축소** | 편관격 + 도화살 충 | 현재 고점, 차익 실현 시기 | 🟢 초록 뱃지 |
| **관리종목** | 공망 진입 + 편인 과다 + 도화살 합거 | 일시 거래 정지 상태 | ⚫ 회색 뱃지 |

#### 2-2. 종목 리포트 카드 레이아웃

**설계 원칙**: 수배 전단처럼 **큰 숫자 하나 + 한 줄 판정 + 차트**로 3초 안에 읽히는 구조. 상세 분석은 카드 아래 접힌 영역에.

```
┌──────────────────────────────┐
│                              │
│  📊 사주증권 리서치센터       │
│                              │
│  김○○의 연애운               │
│  섹터: 도화살 관련주          │
│                              │
│  현재가                      │
│      1,200원                 │ ← 이 숫자가 카드의 중심
│                              │
│  적정가 20,000원  ▼94%      │
│                              │
│  ━━ 2026 차트 ━━            │
│       ╱╲                     │
│  ╱╲╱  ╲  ╱╲╱╲             │
│ ╱      ╲╱      ╱           │
│  NOW→        7월 🚀         │
│                              │
│  투자의견: ★ 강력 매수 ★     │
│                              │
│  "적정가 2만원짜리가         │
│   1,200원에 팔리고 있다."   │
│                              │
│  nadaunse.com/stock          │
└──────────────────────────────┘
```

**카드 구성 요소** (6개 — 3초 안에 읽히는 양):

| 요소 | 역할 | 공유 기여 |
|------|------|----------|
| **현재가 (큰 숫자)** | 카드의 중심 — 시선 고정 | "나 1,200원이래ㅋㅋ 너는?" |
| **적정가 + 저평가율** | 한 줄로 괴리 표현 | "적정가 2만원인데 94% 저평가ㅋㅋ" |
| **차트 + 급등 시점** | 시각적으로 "7월에 오른다" | "7월에 급등하네" |
| **투자의견** | 한 단어로 상태 요약 | "강력매수래ㅋㅋ" "관리종목이래..." |
| **한 줄 코멘트** | 캡처 트리거 — 찌르거나 웃김 | "적정가 2만원짜리가 1,200원에 팔리고 있다" |

#### 2-3. 현재가 산정 로직

**현재가 = 기본가(5,000원) - 억압 감소분 + 모멘텀 가산분**

현재 사주에서 연애운을 억압하는 요소가 많을수록 현재가가 낮아진다. **현재가가 낮을수록 = 저평가 폭이 크다 = 상승 여력이 크다**는 역설이 핵심. 800원 받아도 기분 나쁘지 않음 — "바닥이니까 강력매수"니까.

```
억압 감소분 (사주에서 산출):
  편인 과다 (자격지심)        -1,500원
  도화살 합거 (매력 봉인)     -1,200원
  정관 약 (인연 부재)         -1,000원
  공망 진입                   -800원
  상관 과다 (자기 의심)       -600원

모멘텀 가산분 (현재 대운/세운):
  도화살 운 진입 예정(3개월 내)  +1,500원
  정관 운 진입 예정(6개월 내)    +800원
  편인 합거 예정               +600원
  식신 운 진행 중              +300원
```

| 현재가 구간 | 등급 | 카드 스타일 |
|-----------|------|-----------|
| **8,000원+** | 우량주 | 골드 테두리, "프리미엄 종목" 뱃지 |
| **5,000~7,999원** | 중형주 | 실버 테두리 |
| **2,000~4,999원** | 소형주 | 기본 테두리, "저평가 발굴" 뱃지 |
| **800~1,999원** | 동전주 | 네온 레드 테두리, "역대 저점" 뱃지 |
| **800원 미만** | 관리종목 | 회색 테두리, "관리종목 지정" 경고 |

> 역대 저점(800~1,999원)이 가장 화려한 네온 레드 → 공유 욕구 최대. "관리종목"은 희귀 등급으로 화제성 극대.

#### 2-4. 적정가 산정 로직

현재가와 별개로 **사주가 말하는 진짜 값어치**를 산정하는 것이 핵심 위로 장치. "적정가 20,000원인데 시장가 1,200원" — 이 한 줄이 위로의 전부.

```
적정가 = 기본가(10,000원) + 매력 가산 + 내면 가산 + 역량 가산

매력 가산 (도화살·홍염살·편재 조합):
  도화살 1개                    +3,000원
  도화살 2개                    +7,000원
  홍염살 보유                   +2,000원
  편재 투출                     +1,500원

내면 가산 (정인·식신·비견 조합):
  정인+식신 동주                +3,000원
  비견 적당                     +1,500원
  정관+정인 동주 (진심의 무게)   +4,000원

역량 가산 (편관·정관·편인 조합):
  정관격                        +3,000원
  편관격                        +2,500원
  편인+정관 (커리어형)           +2,000원

적정가 범위: 12,000원 ~ 35,000원
```

| 적정가 구간 | 종목 등급 | 카드 표시 |
|-----------|----------|----------|
| **25,000원+** | 블루칩 | "블루칩 종목" 골드 뱃지 |
| **18,000~24,999원** | 우량주 | "우량주" 실버 뱃지 |
| **12,000~17,999원** | 성장주 | "성장 잠재력" 뱃지 |

**핵심 설계**: 적정가는 대부분 높게 나오도록 설계. 현재가가 1,200원이어도 적정가는 20,000원 — **"저평가율이 높다 = 값어치는 높은데 시장이 몰라본다"**. 현재가와 적정가의 괴리가 클수록 위로가 강해지고, 동시에 조작단의 작전 필요성이 설득력을 가짐.

#### 2-5. 목표가 + 급등 시점 산정 로직

사주 대운·세운에서 **억압 요소가 풀리는 시점**을 급등 시점으로 환산:

```
주요 판단 기준:
  편인이 합거되는 시점     → "자격지심 해소 = 상승 모멘텀"
  도화살 운 진입 시점      → "매력 발현 = 급등"
  정관이 투출되는 시점     → "인연 출현 = 목표가 도달"

목표가 = 적정가의 80~100% 수준
         (현재가 대비 300~1500% 상승률 범위)

표시: "7월 도화살 진입 시 목표가 18,000원 (↑1,400%)"
```

#### 2-6. 투자의견별 애널리스트 코멘트

카드에 박히는 핵심 문장. **이 코멘트가 캡처·공유의 트리거.**

| 투자의견 | 애널리스트 코멘트 | 공유 포인트 |
|---------|----------------|-----------|
| 강력 매수 | "펀더멘털 대비 극심한 저평가. 시장이 이 종목을 모르고 있다." | "시장이 나를 모른대ㅋㅋ" |
| 매수 | "안정적 상승 추세 진입. 중장기 보유 추천." | "꾸준히 오른대" |
| 보유(존버) | "현재 구간에서 손절은 최악의 선택. 존버가 답." | "존버하래ㅋㅋ" |
| 비중 축소 | "고점 근접. 차익 실현 권고." | "나 고점이래 팔아야 하나" |
| 관리종목 | "일시적 거래 정지 구간. 반등은 옵니다." | "관리종목ㅋㅋㅋ 상폐 직전" |

### Step 3. 주가 조작단 소집 (핵심 인터랙션)

종목 리포트 카드 확인 후 **"이 주가, 이대로 두실 겁니까?"** 버튼으로 진입. 3명의 주가 조작단원이 소집되어 턴제 토론이 시작된다.

#### 3-1. 주가 조작단 캐릭터

| 캐릭터 | 포지션 | 투자 철학 | 말투 | 사주 담당 영역 |
|--------|--------|----------|------|--------------|
| **강도현** | 작전 본부장 (공격파) | "비상장이 문제. 시장에 나가야 주가가 움직인다" | 반말, 직설, 급함 | **도화살·편관** (매력·행동력) |
| **윤서율** | 펀더멘털 분석가 (가치파) | "적정가가 높아도 본인이 자기 값어치를 모르면 시장가는 바닥" | 존댓말, 차분, 날카로움 | **편인·정인** (자존감·내면) |
| **차민혁** | 차트 전략가 (타이밍파) | "타이밍이 전부. 7월 전에 준비 안 끝나면 1년 더 기다려야" | 반존대, 계산적, 긴급함 | **대운·세운** (시점·타이밍) |

**캐릭터 간 긴장 구도**:

```
       강도현 (공격파)
      "당장 나가"
        ╱     ╲
       ╱  유저  ╲      ← 유저는 심판석
      ╱  (선택)  ╲
윤서율 ──────── 차민혁
(가치파)        (타이밍파)
"먼저 올려"     "때를 기다려"
```

- **강도현 vs 윤서율**: "당장 소개팅 나가라" vs "자존감도 못 올렸는데 나가면 또 상처만" → 행동 vs 내면 논쟁
- **윤서율 vs 차민혁**: "자기 가치 인식부터 바꿔야 한다" vs "7월 전에 안 끝나면 기회 날린다" → 완벽주의 vs 현실주의 논쟁
- **차민혁 vs 강도현**: "타이밍 안 맞으면 다 소용없다" vs "타이밍 재다가 상폐된다" → 신중 vs 과감 논쟁

**왜 3명이 싸워야 하는가**: 유저를 직접 찌르면 불쾌. 캐릭터끼리 싸우는 걸 보면서 **"ㅋㅋ 맞아 저 말이 맞는데"** 편들기 — 관전자 포지션에서 자연스럽게 자기 문제를 인식하게 됨. 시라노 연애 조작단이 "네가 문제야"라고 말하는 게 아니라, 서로 "이 작전이 맞다" 싸우는 과정에서 유저의 문제가 드러나는 구조.

#### 3-2. 소집 연출

```
━━━━━━ 긴급 소집 ━━━━━━

📊 사주증권 특별 보고

김○○ 종목이
현재가 1,200원까지 하락했습니다.

적정가 20,000원 대비
저평가율 ▼94%.

이대로 두면 관리종목 지정,
최악의 경우 상폐 가능성.

주가 조작단을 소집합니다.

━━━━━━━━━━━━━━━━━━━━

[조작단원 입장]

강도현 (작전 본부장):
"상폐 직전이라고? 종목 정보 줘."

윤서율 (펀더멘털 분석가):
"원국부터 보겠습니다."

차민혁 (차트 전략가):
"대운·세운 차트 띄워주세요."

━━━━━━━━━━━━━━━━━━━━

[작전 회의 시작하기 →]
```

#### 3-3. 작전 회의 — 턴제 토론 (4~5턴)

**1턴: 현황 진단 — "왜 바닥인가"**

세 캐릭터가 각자의 관점에서 저평가 원인을 진단. 서로 다른 원인을 지목하며 충돌.

```
강도현 (작전 본부장):
"원인은 간단해. 비상장이야.
 도화살이 2개나 있는데
 시장에 나가질 않으니까
 아무도 이 종목을 모르는 거야.

 비상장 종목을 누가 사냐고."

윤서율 (펀더멘털 분석가):
"본부장님, 그게 아닙니다.
 이 종목의 적정가를 보세요. 20,000원.
 근데 시장가는 1,200원.

 偏印이 月干에서 일간을
 24시간 누르고 있어요.
 '나 같은 게' '내가 뭐라고'
 이 자기 평가가 시장가를
 바닥으로 끌어내린 겁니다.

 상장해봤자 본인이 1,200원이라고
 믿고 있으면 시장도 그 값에 사요."

차민혁 (차트 전략가):
"둘 다 틀렸어요.
 차트를 보세요.

 지금 丁巳 대운에서
 도화살이 합거 상태예요.
 이건 종목 자체의 문제가 아니라
 시장 환경이 안 맞는 겁니다.

 7월에 丙午 운이 들어오면
 偏印 합거가 풀리면서
 모멘텀이 살아나요.

 지금 아무리 작전 짜봤자
 타이밍 아니면 소용없습니다."
```

유저 선택:
```
누구 말이 맞다고 생각하세요?

  ① 강도현: "맞아, 나가봐야 알지"
  ② 윤서율: "솔직히 자신감이 문제인 건 맞아..."
  ③ 차민혁: "타이밍이 안 맞았던 건가"
```

**2턴: 작전 충돌 — "어떻게 올릴 것인가"**

유저의 1턴 선택에 따라 토론 방향이 갈린다. 유저가 ②번(윤서율)을 선택한 경우:

```
윤서율 (펀더멘털 분석가):
"역시 본인도 아시는군요.

 제 분석으로는 이 종목의
 적정가는 20,000원입니다.
 도화살 2개에 정관+정인 동주.
 펀더멘털은 우량주예요.

 근데 지금 시장가가 1,200원.
 적정가의 6%밖에 안 되는 거예요.
 자기 자신을 저평가하는 습관이
 시장가를 바닥으로 끌어내린 겁니다."

강도현 (작전 본부장):
"서율아, 적정가 적정가 하는데
 적정가가 높으면 뭐해?
 거울 보면서 '나는 괜찮아'
 백 번 외치면 시장가가 올라가냐?

 시장에 나가서 반응을 받아봐야
 '아 내가 괜찮구나'를 아는 거지.

 당장 이번 주 소개팅 잡아.
 실전 매매가 답이야."

윤서율 (펀더멘털 분석가):
"본부장님, 또 그러시는 거예요?
 시장가 1,200원인 상태에서
 시장에 나가면 어떻게 되는지 아세요?

 '역시 안 되는구나.'

 이 한마디가 시장가를 800원까지
 끌어내립니다. 그러면 진짜 상폐예요."

차민혁 (차트 전략가):
"그래서 제가 타이밍이라고 한 거예요.

 지금 나가면 윤서율 말대로 리스크.
 안 나가면 강도현 말대로 기회 없음.

 절충안 하나 내겠습니다.

 4~5월: 자기 가치 인식 기간 (윤서율 작전)
 6월: 시장 노출 시작 (강도현 작전)
 7월: 도화살 진입 시 풀베팅

 시간표가 있어야 작전이지,
 아무 때나 하면 도박이에요."
```

유저 선택:
```
당신의 작전은?

  ① 강도현 안: "바로 시장에 나간다"
  ② 윤서율 안: "내 값어치부터 알아야지"
  ③ 차민혁 안: "시간표대로 단계별로"
```

**3턴: 핵심 찌르기 — "진짜 원인"**

유저의 선택과 무관하게, 이 턴에서 세 캐릭터가 **하나의 핵심 원인**에 수렴한다. 서로 싸우다가 결국 같은 지점을 가리키는 순간 — 유저가 "아..." 하는 턴.

```
강도현 (작전 본부장):
"잠깐. 하나 물어볼게.

 혹시 좋아하는 사람한테
 먼저 연락한 적 있어?"

  ① "...없어"
  ② "했는데 안 됐어"
  ③ "좋아하는 사람이 없어"

→ ①번 선택 시:

강도현:
"그게 비상장이라는 거야.
 종목을 상장을 안 했는데
 누가 사."

윤서율:
"본부장님 말이 맞지만
 이유가 다릅니다.

 연락 못 한 게 아니라
 안 한 거죠?

 '내가 연락하면 부담일까봐.'
 '읽씹당하면 어쩌지.'

 偏印이 하는 일이 이겁니다.
 행동 직전에 브레이크를 거는 거예요.

 적정가 20,000원짜리가
 1,200원에 거래되는 이유가 이거예요.
 매력이 없는 게 아니라
 매력을 꺼놓은 겁니다."

차민혁:
"원국 다시 보겠습니다.

 ...여기 보세요.
 日支 正官에 正印 동주.

 이 구조는 한번 마음을 주면
 전부를 주는 사람이에요.

 근데 偏印이 그걸 막고 있으니까
 시장에는 '관심 없는 사람'으로
 보이는 겁니다.

 실제로는 관심이 폭발하는데
 표면에 안 나오는 종목.

 이런 종목이 제일 아까운 거예요."
```

> 3명이 싸우다가 하나로 수렴하는 이 순간이 **감정 최고조**. "매력이 없는 게 아니라 꺼놓은 거"라는 진단이 세 캐릭터의 입에서 동시에 나오면서 설득력 폭발.

**4턴: 작전 설계 — "3단계 주가 상승 작전"**

핵심 원인이 수렴된 후, 세 캐릭터가 역할을 나눠 작전을 설계한다. 이 턴에서 싸움이 협력으로 전환.

```
강도현:
"좋아. 원인은 파악됐어.
 이제 작전 짜자.

 나는 3단계 담당할게.
 시장 진출 작전."

윤서율:
"저는 1단계요.
 자기 가치 인식부터."

차민혁:
"2단계는 제가.
 모멘텀 확보 — 타이밍 세팅.

 마지막으로 하나만 여쭤볼게요."
```

유저 선택 (작전 계획서에 반영):
```
주가가 올랐을 때, 어떤 모습이길 원하세요?

  ① "좋아하는 사람에게 먼저 말 거는 사람"
  ② "내 가치를 아는 사람이 찾아오는 것"
  ③ "연애든 뭐든 자신감 있는 사람"
```

#### 3-4. 핵심 설계 원칙

| 원칙 | 이유 |
|------|------|
| **캐릭터끼리 싸우고, 유저는 심판** | 직접 찌르면 불쾌. 관전하면서 "ㅋㅋ 맞아 저 말이 맞는데" 편들기가 더 몰입됨 |
| **싸우다가 하나로 수렴** | 3명이 다른 말 하다가 같은 결론에 도달하면 설득력이 3배. "매력을 꺼놓은 거"가 3명 입에서 나옴 |
| **선택지는 2~3개, 최대 4~5턴** | 길면 이탈. 5~7분 내 끝나야 함 |
| **유저의 선택이 작전 계획서에 반영** | "먼저 말 거는 사람" 선택 → 작전 3단계에 "공격적 시장 진출" 배치 |
| **문제 인식 → 해결 방향 제시 → 상세는 잠금** | 챗봇 전환을 위한 호기심 설계 |

### Step 4. 작전 계획서 카드 (1장 캡처 — 전환 엔진)

턴제 토론이 끝나면 **작전 계획서 카드 1장**이 생성된다. 작전 계획서는 공유용 카드가 아닌 **전환용 카드** — "궁금해서 눌러야 하는" 구조가 핵심.

**설계 원칙**: 작전 상세는 카드에 담지 않는다. 카드는 **"이만큼 오를 수 있다"는 숫자 + "근데 방법은 잠겨있다"는 호기심**만 전달.

#### 4-1. 작전 계획서 카드 레이아웃

```
┌──────────────────────────────┐
│                              │
│  📋 주가 상승 작전 계획서     │
│                              │
│  종목: 김○○의 연애운         │
│                              │
│  1,200원 ──→ 18,000원       │
│         ↑1,400%              │
│                              │
│  ── 작전 로드맵 ──           │
│                              │
│  4월  ■■□□□□□□  자기 인식   │
│  6월  □□□□■■□□  모멘텀 확보 │
│  7월  □□□□□□■■  급등 🔒     │
│                              │
│  ── 조작단 한마디 ──         │
│                              │
│  "적정가 20,000원짜리가      │
│   1,200원에 팔리고 있다.     │
│   이건 시장의 실수다."       │
│                              │
│  🔒 3단계 작전은 1:1로만     │
│                              │
│  nadaunse.com/stock          │
└──────────────────────────────┘
```

**카드 구성 요소** (6개 이하 — 3초 안에 읽히는 양):

| 요소 | 역할 | 공유 기여 |
|------|------|----------|
| **현재가 → 목표가** | 한 줄로 상승 여력 | "1,200원에서 18,000원까지 간대ㅋㅋ" |
| **↑1,400%** | 큰 숫자 하나가 시선 고정 | "1,400% 상승 여력이래" |
| **작전 로드맵 바** | 3단계가 시각적으로 보임 | "7월에 급등인데 잠겨있어" |
| **조작단 한마디** | 캡처 트리거 — 짧고 찌르는 한 줄 | "시장의 실수래ㅋㅋ" |
| **🔒 잠금** | 호기심 유발 | "3단계가 뭔데?" → 전환 |

> 작전 상세(편인 해체, 도화살 타이밍 등)는 카드가 아니라 **턴제 토론 과정**에서 이미 전달됨. 카드는 결론만.

#### 4-2. 3단계 잠금이 전환을 만드는 구조

| 요소 | 역할 |
|------|------|
| **숫자(↑1,400%)** | "이만큼 오를 수 있다니?" → 기대감 |
| **로드맵에서 7월만 🔒** | 시각적으로 "마지막만 빠져있다" |
| **조작단 한마디** | 토론에서 감정이입한 캐릭터의 말 → 신뢰 |
| **"1:1로만"** | 비밀 작전 느낌 → 호기심 → 클릭 |

> **핵심**: 토론에서 문제를 인식시키고, 카드에서 "답이 있다"를 보여주되, 실행은 잠금. "마지막 한 발이 빠져있다"는 미완의 구조가 챗봇 전환의 엔진.

### Step 5. 커플 M&A 시너지 분석 (강제 2인 루프)

작전 계획서 확인 후 **"합병 대상 종목 분석"** 화면으로 전환. **상대방 생년월일이 있어야 분석 가능** — 제품 자체가 공유 없이 작동 불가한 구조.

#### 5-1. M&A 분석 입력

상대 종목명(이름) + 종목코드(생년월일) 입력. 또는 **"상대에게 종목코드 요청하기"** → 카카오톡 링크 전송.

#### 5-2. M&A 리포트 카드

**설계 원칙**: 이름궁합 테스트처럼 **큰 숫자 하나 + 한 줄 판정**으로 끝. 시너지 내역·리스크 상세는 카드 아래 접힌 영역에.

```
┌──────────────────────────────┐
│                              │
│  🤝 합병 시너지 분석          │
│                              │
│  김○○  ×  이○○              │
│                              │
│       +30%                   │ ← 이 숫자가 카드의 전부
│    합병 시너지                │
│                              │
│  ★★★★☆  합병 추천           │
│                              │
│  "이 조합, 같이 있으면       │
│   서로의 주가가 오릅니다."   │
│                              │
│  ⚠️ 충돌 확률 23%            │
│                              │
│  nadaunse.com/stock          │
└──────────────────────────────┘
```

**카드 구성 요소** (5개 — 3초 안에 읽힘):

| 요소 | 역할 | 공유 기여 |
|------|------|----------|
| **이름 × 이름** | 누구와 누구인지 한눈에 | "우리 합병 분석했는데ㅋㅋ" |
| **+30% (큰 숫자)** | 카드의 핵심 — 시선 고정 | "우리 시너지 30%래" "우리 마이너스래ㅋㅋ" |
| **별점** | 직관적 판정 | "별 4개래" |
| **한 줄 코멘트** | 캡처 트리거 | "같이 있으면 주가 오른대ㅋㅋ" |
| **충돌 확률** | 긴장 요소 — 논쟁 유발 | "근데 충돌 23%래" |

> 시너지 상세 내역(정서적/성장/사회적)은 카드 아래 **"상세 보기"** 접힌 영역으로. 카드 자체는 숫자 하나 + 한 줄.

- **1명이 여러 명과 M&A 분석 가능** → "짝사랑, 썸남, 전남친 다 넣어보기" → 바이럴 배수
- "+30%" 숫자가 비교 대화 유발 → "우리 시너지 몇 %야?"
- 마이너스가 나오면 더 웃김 → "합병하면 주가 떨어진대ㅋㅋ" → 자조 공유

#### 5-3. M&A 전송 메시지

```
┌─ 카카오톡 공유 메시지 ──────────┐
│                                  │
│  📊 [합병 시너지 분석 요청]      │
│                                  │
│  [OO]님이 당신과의 합병          │
│  시너지를 분석하고 싶어합니다.   │
│                                  │
│  생년월일만 입력하면 끝!         │
│                                  │
│  [시너지 확인하기 →]             │
│                                  │
└──────────────────────────────────┘
```

- 받는 사람: "합병 시너지 분석 요청왔는데ㅋㅋ" → 링크 클릭 → 자기도 시세 조회 → **K-factor 2+**

### Step 6. 유료 전환 CTA — 작전 실행

작전 계획서의 **잠긴 3단계**와 캐릭터 선택이 전환 동기.

#### 6-1. 캐릭터별 전환 멘트

| 캐릭터 | CTA 버튼 | 전환 멘트 |
|--------|---------|----------|
| **강도현** | "강도현과 작전 실행하기" | "3단계는 여기 못 써. 1:1로 와. 7월에 정확히 뭘 해야 하는지 알려줄게." |
| **윤서율** | "윤서율과 1단계 시작하기" | "1,200원을 적정가까지 올리는 거, 혼자는 어렵습니다. 같이 해요." |
| **차민혁** | "차민혁과 타이밍 잡기" | "7월까지 남은 시간, 주 단위로 짜드리겠습니다. D-day 역산 들어갑니다." |

#### 6-2. 전환 플로우

```
작전 계획서에서 캐릭터 선택
    ↓
CTA 버튼 클릭
    ↓
캐릭터 챗봇 페이지로 딥링크
    ↓
캐릭터가 작전의 연장선에서 대화 시작:

강도현: "왔어? 좋아. 3단계 얘기하자.
        리포트에 쓸 수 없었던 건..."

윤서율: "오셨군요. 시장가 올리기,
        오늘부터 시작합니다.
        먼저 하나만 여쭤볼게요."

차민혁: "D-97. 7월까지 남은 날입니다.
        주 단위 작전 들어갑니다."
    ↓
유료 1:1 상담 진행 (350포인트)
```

- 토론에서 이미 캐릭터에 감정이입 → 전환 허들 최소화
- 작전 계획서의 사주 내용을 챗봇이 이어받아 상담 → 맥락 연결
- **3단계 잠금이 만든 호기심**이 전환의 핵심 엔진

### Step 7. 공유 + 재방문

#### 7-1. 공유 옵션 (종목 리포트 카드 + 작전 계획서)

```
┌──────────────────────────────┐
│  [📱 카카오톡으로 보내기]     │
│  [📸 인스타 스토리에 올리기]  │
│  [🔗 링크 복사]              │
│  [💾 이미지 저장]            │
│                              │
│  [🤝 M&A 시너지 분석하기]    │
│  → "관심 종목과 합병 분석"    │
│                              │
│  [🔄 다른 종목 분석하기]     │
│  → "친구 종목코드 넣어서      │
│     리포트 뽑기"              │
└──────────────────────────────┘
```

#### 7-2. "남의 사주로 종목 리포트 뽑기" (확산 가속기)

- 본인 결과 확인 후 → "친구/썸남 종목도 분석해볼래?" 유도
- 상대방 생년월일 입력 → 상대의 종목 리포트 생성 → "야 너 현재가 이거래ㅋㅋ" 공유
- **1인이 N개 종목 리포트를 뽑는 구조** → 바이럴 배수 효과

#### 7-3. 재방문 트리거

| 트리거 | 내용 |
|--------|------|
| **주간 시황 리포트** | 매주 월요일 "이번 주 연애운 시황" 자동 생성 → "수요일 급등 주의보" 일별 예보 → 재방문 |
| **급등 알림** | 급등 예정 월 접근 시: "📈 7월 급등 시그널 감지. 작전 준비 상태를 점검하세요." → 재방문 |
| **조작단 속보** | "강도현: 3단계 시장 상황이 바뀌었습니다. 작전 수정 필요." → 캐릭터 푸시로 재방문 |
